Opcion Valor Descripcion
AutocompletadoSi esto es activado, se generarán consultas para sugerir un autocompletado donde corresponda (por ejemplo al escribir un nick en los mensajes privados), esto genera una carga extra en el servidor/script, por lo que si tiene problemas de rendimiento recomendamos desactivarlo.
Limite de busquedaEstablece el limite de resultados de una busqueda (cuanto más grande, más consumo de recursos habrá).
Buscar relacionadosSi esto se activa, el script buscará relacionados al entrar a un post, (utilizando el titulo como patrón de busqueda), si esto es desactivado, el script obtendrá al azar una cantidad de post de la misma categoría para mostrarlos. Tenga en cuenta que la busqueda en cada post genera una carga importante sobre el server/script.
Cantidad de relacionadoscuanto más grande este numero, más carga sobre el server habrá, pero la lista de relacionados (dentro de los post) será proporcionalmente más grande.
Recachear relacionadosEstablece el tiempo antes de recachear relacionados en segundos, esto reduce conciderablemente el consumo de recursos cuanto más tiempo se establezca para recachear.
Recachear postsSi esto se activa, se ignorará el caché de bbcodes de los post, esto aumenta considerablemente el consumo de recursos inneceseariamente.
